Prueba técnica del proyecto TbDivisa.

Este proyecto, es una aplicación JAVA para el cálculo del valor de una TbDivisa en otra

Para el desarrollo del proyecto se hizo uso de
JDK:jdk1.8.0_221
Application Server: Payara 5.193.1
IDE: NetBeans 11.2
Gestión y construcción del proyecto: Maven 3.3.9
Motor de base de datos: potsgresql

La descripción del proyecto se encuentra en el archivo 
Prueba_Tecnica_STA.pdf

Base de datos se encuentra en el proyecto.
TbDivisadb.sql
